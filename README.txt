----------------
Mail Merge
----------------
Customize the letter: Go to Input->Letters->starting_letter.txt and edit the file. Do not change the first line!
Customize the names: Go to Input->Names->invited_names.txt and edit the file. One name per line!

Copying the letters: Double-click main.py. This will copy your letters to Output->ReadyToSend
